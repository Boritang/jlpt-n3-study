# JLPT N3 Kanji Study App

Static mobile study app for JLPT N3 vocabulary and kanji mnemonics.

## GitHub Pages

Upload `index.html` to a GitHub repository and enable Pages from the repository root.

Recommended repository name:

```text
jlpt-n3-study
```

After Pages is enabled, the app will be available at:

```text
https://YOUR_GITHUB_USERNAME.github.io/jlpt-n3-study/
```
